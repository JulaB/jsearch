# Json file search

## Features
* exact search. Ex: "some phrase"
* contains search. Ex: several words
* exclude search with '-'. Ex: -search-word
* exact exclude search. Ex: -"Some phrase"

## Technologies
* Ruby 2.6.0
* Rails 5.2.2.1
* Minitest

## Install
* yarn install
* bundle install
* rails test

## Development
To run server in development `rails server`

## Running tests
* rails test
